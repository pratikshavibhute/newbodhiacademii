What is mobile application development??

mobile application development?

In web development we build web applications. A web application is an application that is rendered on the browser, and accessed by the user through the browser. Simply put - when you order tickets to movie in your home town theatre, when you check your bank account balance, when you write a post on Facebook or update your resume on LinkedIn, all of these and more are examples to web applications.

You open a web browser, go to a specific URL, and get a full working application. Today you can even find sophisticated applications on the web, like sound and video editing applications, drawing, recording, gaming, word processing, you name it. They're all on the web, and all are accessed via the browser.
